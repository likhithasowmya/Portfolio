Telugu MRC dataset Creation Tool

The tool we propose includes a user-friendly GUI and a backend that performs web scraping using Python libraries like BeautifulSoup and requests, and data processing using Pandas. The GUI allows the user to select the Telugu Wikipedia webpages to be scraped and specify the number of questions to be generated from each page. The backend extracts relevant sentences from the webpages and generates questions based on the content. The questions are stored in a CSV (Comma-Separated Values) file for further analysis and training of machine learning models..
